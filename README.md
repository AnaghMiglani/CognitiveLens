This project is a real-time cognitive analytics system designed for online classroom environments, where a teacher can monitor student engagement through visual cues captured via a webcam. The system processes live video input and applies computer vision techniques using MediaPipe facial landmark detection to extract meaningful behavioral signals. It consists of multiple independent modules that analyze different aspects of a student’s state, including drowsiness, attention, stress, and confusion. Each module operates on the same video frame but computes distinct metrics based on facial features such as eye aspect ratio (EAR), mouth aspect ratio (MAR), head pose, eyebrow movement, and blink rate.

The drowsiness module detects signs of fatigue using a combination of eye closure and yawning patterns, outputting a binary score indicating whether the student is sleepy. The attention module tracks head orientation and movement stability, measuring how frequently the student looks away from the screen and converting it into an attention score. The stress module evaluates facial tension through eyebrow compression, lip tightening, and blink frequency to produce a stress score. The confusion module analyzes more subtle cues such as eyebrow asymmetry, inward brow movement, slight squinting, head tilt, and repeated side glances to estimate a confusion score.

All these modules are integrated into a unified dashboard that displays six synchronized video panels: raw camera feed, facial landmarks, and outputs from each cognitive detector. Alongside this visual interface, the system includes a separate analytics window built using PyQt and PyQtGraph, which plots real-time graphs of drowsiness, attention, stress, and confusion over time. These graphs provide a continuous view of the student’s cognitive state, enabling instructors to observe trends and identify moments of disengagement or difficulty.

The architecture is modular, allowing each detector to independently process frames and return both annotated visuals and numerical scores. These scores are aggregated and streamed into the analytics layer without interrupting the video processing pipeline. The system is optimized for real-time performance and designed to be extensible, making it suitable for scaling to multiple students or integrating additional behavioral metrics. Overall, the project demonstrates how computer vision can be applied to enhance digital learning environments by providing actionable insights into student engagement and well-being.
